dmx_exec_t* decodeDMX ( uint8_t  values[])

Decodes the fetched web values.

Decode the fetched values and creates a dmx_exec_t structure that represents the transition from the current state to the one described by the fetched values. NB: If the transition fade time interval is out of range, it is set to 1.

Parameters
valuesthe fetched web values.
Returns
Returns the dmx_exec_t struct object describing the transition.
