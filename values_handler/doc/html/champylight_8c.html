Detailed Description

Main file for the values handler.

Author
Louis Blin
Date
June 2015

This simple program runs on a device connected to the spotlights. It handles the values recovery from the web, and the runs the transitions between light states.

Code using the modified version of the dmx.h library used by the DMXWHEEL project - www.dmxwheel.com.

See also
www.dmxwheel.com

int main ( int  argc,

Entry point of the values handler.

Aim of the main function:

  • init the dmx connection on start.
  • run fetch-decode-execute cycles in order to update the DMX values. NB: if fetched values are the same as on the last cycle, the decode-execute part of the cycle is dropped.
  • closes the dmx connection on exit.
